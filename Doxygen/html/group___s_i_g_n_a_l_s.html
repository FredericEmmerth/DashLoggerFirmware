Detailed Description

This Module Handles the Information Interpretation Handling of All Inputs to Outputs. Signals have a Type, wich changes how it is interpreted and used. For more on this, refer to signals_data_type and signals_signal_type. Signals can be chained together. For Example, four Signals of Type SIGNALS_CAN_MESSAGE could recieve four Temperatures, wich would then be interpreted to float Values using thier respective Callback Functions. Another Signal of Type SIGNALS_INTERNAL_SIGNAL could then find the highest of these Temperatures using its Callback Function. To complete the Chain, a Signal of Type SIGNALS_DISPLAY_SIGNAL can be used to convert the float value of the previous Maximising Function to a String Command wich can be interpreted by a Display.
