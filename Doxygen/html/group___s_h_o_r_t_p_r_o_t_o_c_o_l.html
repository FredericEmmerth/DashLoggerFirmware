Detailed Description

This Module handles packing the Commands generated by the SIGNAL Module and sending them using the Shortprotocol defined in the Displays (EA uniTFTs035-ATC) Datasheet. To not block the Main Loop for too long, the Packet is split into smaller Parts defined by SHORTPROTOCOL_Instance.maximumPackageLength. The Module is written in a general Fashion and can be adopted to use SPI, for this the Callback Functions in SHORTPROTOCOL_Instance have to be redefined for SPI.
