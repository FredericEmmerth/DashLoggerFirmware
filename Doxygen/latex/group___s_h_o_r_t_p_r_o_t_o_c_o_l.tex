\doxysubsection{Detailed Description}
This Module handles packing the Commands generated by the SIGNAL Module and sending them using the Shortprotocol defined in the Displays (EA uni\+TFTs035-\/\+ATC) Datasheet. To not block the Main Loop for too long, the Packet is split into smaller Parts defined by \mbox{\hyperlink{struct_s_h_o_r_t_p_r_o_t_o_c_o_l___instance_a38a5e509c9750876370dbfd98ea63680}{SHORTPROTOCOL\+\_\+\+Instance.\+maximum\+Package\+Length}}. The Module is written in a general Fashion and can be adopted to use SPI, for this the Callback Functions in \mbox{\hyperlink{struct_s_h_o_r_t_p_r_o_t_o_c_o_l___instance}{SHORTPROTOCOL\+\_\+\+Instance}} have to be redefined for SPI.
