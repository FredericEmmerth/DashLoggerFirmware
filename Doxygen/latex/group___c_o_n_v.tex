\doxysubsection{Detailed Description}
In this Module the Callback Functions for the Signals from the \mbox{\hyperlink{group___s_i_g_n_a_l_s}{SIGNALS}} Module are defined.

There are Callback Functions for CAN Message Interpretation, for Internal Message Processing and Display Command Generation.
